## What is LinuxNT?

LinuxNT is not a copy of Microsoft's NT kernel (it's closed source).  
It is a **reimplementation of NT design ideas** on top of the Linux kernel:

- NT's **object manager**, **registry**, **service control**, **subsystem architecture**
- Linux's **security**, **stability**, **open source ecosystem**
